# JavaScript正则表达式

## 一、元字符

### 1、^ 与 $

（1）、以... 开头，以... 结尾

```javascript
let reg = /^\d/
console.log(reg.test('')) //
console.log(reg.test('')) //
console.log(reg.test('')) //
```

```javascript
let reg = /\d$/
console.log(reg.test('')) //
console.log(reg.test('')) //
console.log(reg.test('')) //
```

（2）、两个都不加，包含即可

```javascript
let reg = /\d+/ // 等价于 let reg = /\d/
console.log(reg.test('fdsf2fdal')) // true
console.log(reg.test('232')) // true
console.log(reg.test('2')) // true
```

（3）、两个都加，表示只能是

```javascript
let reg = /^\d$/ // 以数字开头，不知道以什么结尾 => 只能是 一个数字
console.log(reg.test('1')) // true
console.log(reg.test('12')) // false
console.log(reg.test('1u')) // false

let reg = /^\d+$/ // 以数字开头，以数字结尾，中间都是数字 => 只能是 至少一个数字
reg.test('24') // true
reg.test('2444') // true
reg.test('2') // true
reg.test('2334334343') //true

let reg = /^\d\d$/ // 以数字开头，以数字结尾，中间没有任何东西 => 只能是 两个数字
console.log(reg.test('24')) // true
console.log(reg.test('234')) // false
console.log(reg.test('2')) // false
console.log(reg.test('247834')) // false

// 验证11位手机号码
let reg = /^1\d{10}$/ // 只能是 1后面跟10个数字
console.log(reg.test('')) //
console.log(reg.test('')) //
console.log(reg.test('')) //
```
